Key Cave Adventure

Key Cave Adventure Game is a single-player dungeon-crawler game where the player adventurously explores a dungeon. The objective is for the player to find the key and get out of the dungeon through the door. This game is full of adventure.

As part of my university project this is my first python GUI game I have built. With the aid of tkinter and pillow packages I was able to program a playable game.
